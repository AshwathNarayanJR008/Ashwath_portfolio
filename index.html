A Deep Neural Network Approach for Automated Blood Vessel Segmentation, Noise Reduction, and 3D Reconstruction in Ultrasound Images

Current MSc Dissertation Project

My Master's dissertation aims to develop AI-based segmentation model for automatic detection of blood vessel in ultrasound images. The model learns to differentiate between the region of interest (blood vessel) and the background by training on labeled ultrasound images. The ultimate goal is to improve the accuracy and efficiency of medical image analysis, reducing the manual effort required by radiologists and assisting in early diagnosis.

HeartSync: A Vision-Based Real-Time Vital Signs Monitoring System

The project uses a webcam to estimate vital signs like heart rate, respiratory rate, and body temperature in real-time. It uses computer vision and signal processing techniques to detect physiological cues and computes BPM and RPM using Fast Fourier Transform. The system has a modular Python-based GUI for user interaction and is suitable for home-based health tracking and telemedicine. It also includes cloud-based data logging via Google Sheets and is adaptable to mobile platforms. Future work will focus on clinical validation, stress level prediction, and SpO₂ estimation.

BrainMap AI: Hybrid System for Brain Tumour Detection and Segmentation

This project combines supervised and unsupervised learning to detect and segment brain tumours from MRI images without labeled data. A ResNet-18 model classifies brain MRI scans into four views and identifies tumour presence. When a tumour is detected, unsupervised clustering methods (K-means and fuzzy C-means) segment the tumour regions. Two user-friendly GUIs handle classification and segmentation, and the results, along with patient details, are automatically emailed to physicians—making diagnosis easier and more efficient.

Early Prediction of Heart Failure using Machine Learning

The project aims to support early diagnosis and intervention for heart failure patients using artificial intelligence. A heart failure dataset was thoroughly analyzed and preprocessed to ensure data quality. Six different machine learning models were trained and evaluated, with logistic regression selected as the final model due to its strong performance (around 90% in accuracy, precision, and recall). To make the system accessible in clinical environments, a graphical user interface (GUI) was developed, allowing healthcare professionals to input patient data and receive instant predictive results. This project demonstrates how AI can enhance clinical decision-making and improve patient outcomes in cardiovascular healthcare.

Smart Non-Invasive Health Monitoring for Glucose

This project focused on building a real-time health monitoring system using wearable IoT devices to gather patient data non-invasively. A key feature was the development of a glucose identification model, which accurately detected blood glucose levels without invasive methods, offering a convenient solution for diabetic patients. Machine learning algorithms processed the data to provide actionable insights, enabling healthcare providers to make more informed decisions, ultimately improving patient outcomes and reducing the strain on traditional healthcare resources.
